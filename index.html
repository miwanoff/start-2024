HTML+CSS+JavaScript

Всемирная паутина (World Wide Web, WWW) —
сеть, состоящая из связанных между собой документов и других веб-ресурсов, расположенных на различных компьютерах и доступных через Интернет.

HTML, CSS и JavaScript - три основные технологии World Wide Web.

HTML (HyperText Markup Language) — язык гипертекстовой разметки документов во Всемирной паутине. Большинство веб-страниц создаются при помощи языка HTML.

CSS (Cascading Style Sheets) — каскадные таблицы стилей. Формальный язык описания внешнего вида документа. Отвечает за то, как страницы выглядят.

JavaScript (JS) - высокоуровневый, интерпретируемый язык программирования. Делает сайты динамичными и интерактивными.

Самая распростаненная CMS в мире - WordPress

WordPress (content management system, CMS) —
система управления содержимым сайта, с помошью которой можно создавать любые сайты - от простых блогов до сложных новостных порталов и интернет-магазинов.

Самая распростаненная CMS в мире, более трети всех сайтов WWW сделана на ней.
